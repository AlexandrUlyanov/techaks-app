# Яндекс Вебмастер и Метрика: рабочая настройка для Techaks

Дата: 2026-06-04

## 1. Яндекс Вебмастер

### Что нужно настроить

1. Главное зеркало:
   - `https://techaks.ru`
2. Регион:
   - `Пенза`
3. Sitemap:
   - `https://techaks.ru/sitemap.xml`
4. Переобход:
   - главная
   - 3-5 ключевых категорий
   - 3-5 ключевых товаров
   - 2 брендовые страницы
   - акции
   - блог

### Что контролировать

- индексирование категорий;
- индексирование товаров;
- excluded pages;
- конфликты canonical;
- ошибки robots;
- проблемы качества сниппетов.

## 2. Яндекс Метрика

Метрика грузится только после cookie consent.

### Что уже подключено

- базовый счётчик;
- ecommerce через `dataLayer`;
- события:
  - `view_item`
  - `add_to_cart`
  - `begin_checkout`
  - `reserve_item`
  - `purchase`
  - `lead_submit`
  - `order_message`

### Какие цели использовать

- `view_item`
- `add_to_cart`
- `begin_checkout`
- `reserve_item`
- `payment_success`
- `lead_submit`
- `order_message`

## 3. Как проверять

### Проверка ecommerce

1. Открыть страницу товара.
2. Добавить товар в корзину.
3. Перейти в checkout.
4. Сделать резерв товара.
5. Провести тестовую оплату YooKassa.
6. Отправить лид-форму.
7. Отправить сообщение по заказу из личного кабинета.

Ожидаемое поведение:
- события уходят в `dataLayer`;
- цели вызываются через `ym(..., "reachGoal", ...)`;
- до cookie consent Метрика не должна загружаться.

### Проверка на проде

- посмотреть сетевые запросы к `mc.yandex.ru`;
- посмотреть `window.dataLayer`;
- проверить, что повторный заход на `payment/result` не дублирует `purchase` бесконечно.
