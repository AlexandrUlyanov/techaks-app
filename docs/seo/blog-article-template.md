# Шаблон SEO-статьи для ТЕХАКС

## Структура материала

1. **H1**
2. **Короткий ответ**
3. **Вступление с контекстом**
4. **Основной разбор по блокам**
5. **Что открыть по теме**
6. **FAQ**
7. **Связанные статьи / категории / бренды**

## Редакционные правила

- писать по-русски, спокойно и без рекламной воды;
- объяснять через сценарий использования;
- не делать абзацы на 8-10 строк;
- не обещать неподтверждённые характеристики;
- не вставлять фейковые цифры и кейсы;
- не писать ради ключей, а держать естественный коммерческий интент.

## SEO-правила

- `title`: до 255 символов, с главным интентом;
- `metaDescription`: до 320 символов, без дублей;
- `excerpt`: короткая полезная выжимка;
- `slug`: понятный, латиницей, без мусора;
- FAQ: минимум 2 вопроса, если тема это допускает;
- добавить ссылки на категории и бренды, которые реально помогают пользователю.

## Обязательные внутренние переходы

- минимум 2 ссылки на категории;
- минимум 1 ссылка на бренд или брендовый каталог, если это релевантно;
- по возможности 1 ссылка на соседнюю статью кластера.

## Признаки хорошей статьи

- ответ понятен уже в первых абзацах;
- пользователь понимает, куда идти дальше в каталог;
- статья полезна и без покупки, но мягко подводит к выбору;
- SSR-страница выглядит полноценно даже без клиентского JS.
