# SEO-212 — Семантическое ядро, интенты и карта целевых страниц Techaks

## Зачем нужен этот документ

Это рабочая карта для трёх параллельных потоков:

1. storefront SEO;
2. контент и блог;
3. GEO/AEO-ответы под Яндекс и AI-поиск.

Документ не претендует на “полное ядро навсегда”. Его задача — дать нормальную базу, из которой можно:

- планировать SEO-спринты;
- запускать локальные и брендовые страницы;
- делать контент без хаоса и дублирования;
- быстро понимать, какой URL отвечает за какой intent.

## Основные типы запросов

### 1. Товарные

Пользователь уже знает, что ищет:

- купить hoco zp7
- hoco hp65 цена
- яндекс станция мини 3 в пензе

Целевой URL:

- карточка товара `/product/:slug`

### 2. Категорийные

Пользователь выбирает группу товаров:

- наушники в пензе
- зарядные устройства купить
- аксессуары для смартфонов

Целевой URL:

- категория `/catalog?cat=:slug`

### 3. Брендовые

Пользователь хочет товары конкретного производителя:

- hoco в пензе
- remax купить
- samsung аксессуары

Целевой URL:

- бренд `/catalog?view=brands&brand=:slug`

### 4. Локальные коммерческие

Пользователь ищет покупку рядом:

- где купить наушники в пензе
- смартфоны пенза самовывоз
- зарядка айфон пенза

Целевой URL:

- category landing + stores page
- отдельные локальные лендинги при необходимости

### 5. Информационные

Пользователь ещё не выбрал товар:

- как выбрать роутер
- чем отличаются беспроводные наушники
- какой кабель type-c выбрать

Целевой URL:

- блог `/blog/:slug`
- иногда category landing с answer-first intro

### 6. AI-вопросы / answer-first

Запросы под AI-поиск и conversational intent:

- какие наушники есть в наличии в пензе
- где купить зарядку hoco рядом
- какие товары hoco есть в techaks

Целевой URL:

- category / brand / product SSR-страницы с factual blocks, FAQ и внутренними ссылками

## Локальные лендинги первой волны

Эти кластеры можно закрывать либо через уже существующие category pages, либо через отдельные локальные landing pages, если видим хороший спрос.

| Локальный кластер | Базовый target | Когда нужен отдельный landing |
|---|---|---|
| наушники в Пензе | `/catalog?cat=naushniki` | если нужен усиленный локальный answer-first контент |
| зарядки в Пензе | `/catalog?cat=setevye-adaptery` | если хотим расширить long-tail “зарядка iphone/type-c/быстрая” |
| кабели в Пензе | `/catalog?cat=kabeli-i-razvetviteli` | если category page не закрывает интент по разъёмам |
| яндекс станции в Пензе | `/catalog?cat=yandeks-stancii` | отдельный landing не обязателен, category уже сильная |
| роутеры в Пензе | `/catalog?cat=setevoe-oborudovanie` | при появлении контентного спроса “для квартиры/дачи/офиса” |

## Формула target page

Для каждого кластера нужно заранее определить:

1. **Target URL**
2. **Title**
3. **H1**
4. **Short answer-first intro**
5. **FAQ**
6. **Внутренние ссылки**
7. **Наличие локального сигнала**

### Шаблон для category landing

- Title: `{Категория} в Пензе — ТЕХАКС`
- H1: `{Категория}`
- Intro: `Выбирайте {категория} с актуальными ценами, наличием в Пензе и самовывозом в магазинах ТЕХАКС.`
- FAQ:
  - что есть в разделе;
  - можно ли забрать сегодня;
  - как выбрать;
  - есть ли доставка.

### Шаблон для brand landing

- Title: `{Бренд} в Пензе — товары бренда в ТЕХАКС`
- H1: `{Бренд}`
- Intro: `На странице бренда {Бренд} собраны актуальные товары с ценами, наличием и быстрым переходом в нужные категории.`
- FAQ:
  - какие товары бренда есть;
  - можно ли забрать самовывозом;
  - как выбрать;
  - есть ли доставка.

### Шаблон для product page

- Title: `{Товар}, {ключевая характеристика} — купить в Пензе в ТЕХАКС`
- H1: название товара
- Intro: короткий summary + factual block
- FAQ:
  - что важно знать;
  - есть ли наличие в Пензе;
  - можно ли забрать самовывозом;
  - доступна ли доставка.

## Внутренние ссылки: обязательная логика

### Категория должна ссылаться на:

- соседние разделы;
- популярные бренды внутри раздела;
- статьи и подборки по теме;
- stores / delivery, если это усиливает коммерческий intent.

### Бренд должен ссылаться на:

- релевантные категории;
- популярные товары;
- статьи о выборе и совместимости.

### Товар должен ссылаться на:

- категорию;
- доставку и оплату;
- магазины;
- отзывы / review section.

## Что брать следующим спринтом

1. Запустить backlog локальных и брендовых лендингов первой волны.
2. Собрать 5–7 статей knowledge center под кластеры “как выбрать”.
3. Привязать кластеры к weekly review в SEO dashboard.
4. На production/staging регулярно пересобирать shortlist SKU для ручного enrichment.
