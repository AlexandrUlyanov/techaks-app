# Knowledge center для блога ТЕХАКС

## Цель

Сделать блог не витриной случайных статей, а постоянным SEO- и GEO/AEO-слоем:

- закрывать коммерческие и информационные интенты;
- связывать статьи с категориями и брендами каталога;
- формировать кластерную перелинковку для Яндекса;
- вести пользователя из ответа сразу в релевантный раздел каталога.

## Что обязательно должно быть в каждой статье

- answer-first абзац в первых 1-2 экранах;
- связка минимум с 2 категориями каталога;
- связка минимум с 1-2 брендами, если это уместно;
- FAQ-блок из 2-4 вопросов;
- мета title и meta description без дублей;
- чистый slug на латинице;
- внутренняя перелинковка на релевантные разделы;
- пригодность для SSR-индексации и Schema.org.

## Что проверяем после публикации

- страница входит в sitemap;
- SSR-HTML содержит H1, lead, FAQ и внутренние ссылки;
- title и description уникальны;
- статья связана с каталогом, а не живёт отдельно;
- материал не дублирует соседние темы по интенту.
