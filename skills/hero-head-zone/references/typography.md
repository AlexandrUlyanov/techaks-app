# Typography Reference

## Hero Headline Rules

- **Size:** `clamp(3.5rem, 9vw, 8rem)`
- **Line height:** `0.92–1.0`
- **Letter spacing:** `-0.03em` to `-0.05em`
- **Gradient text:** Apply only to 1–3 key words.

## Font Pairings

- **Tech:** Inter (UI) + Space Grotesk (Display)
- **Luxury:** Playfair Display (Head) + Lato (Text)
- **Creative:** Syne / Clash Display (Head) + Satoshi (Text)
